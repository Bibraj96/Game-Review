# Game Review

A site to rate and review video games! You can:

* Add games to review
* Add genres
* CRUD (Create, Read, Update, and Delete) you reviews
* View other users' reviews

## Installation

To user this site, clone this repo and then run `bundle`

## Usage

1) run `rake db:migrate`
2) run `rake db:seed`
3) run `rails s`
